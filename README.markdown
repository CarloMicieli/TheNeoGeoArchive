# TheNeoGeoArchive

A playground application ".NET core all the things" - a tribute/archive to the best gaming console of all time.

## Data Loader

Filling the database using a bunch of csv files.

Inside the `src/DataLoader` directory, just type

```
$ dotnet run
```
